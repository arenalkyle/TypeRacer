# TypeRacer
Terminal-based Type Racing game built with Rust and the Ratatui library. Test your typing speed (WPM) and accuracy directly from your command line.

## Features
**Real-time Stats**: Track your Words Per Minute (WPM) and displays when you finish the game.
**Instant Feedback**: Visual highlighting for correct, incorrect and current characters.
**Terminal-Based**: High performance with minimal resource usage.
**Cross-Platform**: Works on Linux, macOS, and Windows
